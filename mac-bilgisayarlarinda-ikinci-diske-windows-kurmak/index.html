Mac bilgisayarlarında ikinci diske windows kurmak

Anlayışınıza sığınırım artık 🙂

Bu yazımızda Mac sistemlerine (iMac,Macbook Air,Macbook Pro,Mac Mini,Mac Studio) sistemlerinde takılan ikinci diske windows kurulumunu anlatmaya çalışacağım.

Not :Bu işlem için PCI-E ve Sata girişli Mac sistemi bulunması gerekmektedir.

Disk bölümü seçeneğine geldiğimizde Windows’a kısıtlı alan verip Boot Camp ile Windows isosunu yazdırıp Apple Windows Destek yazılımını yüklediğimimizde “Disk İzlencesi” ni açalım.

Diskimizde bölüm olan “BOOTCAMP” bölümünü silip ana diski birleştirelim.

İkinci defa Boot Camp’i açalım ve disk bölüm aşamasına geldiğimizde “Etkinlik Monitörü” üzerinden bootcamp’i sonlandıralım.

Ayarlar-> Genel -> Başlangıç Diski aşamalarını takip ederek Windows’u seçerek “Yeniden Başlat” butonuna tıklıyoruz.

Windows kurulum aşamasına geldiğimizde disk seçim aşamasına gelip;

Klavyeden Shift ve F10 tuşlarına basıyoruz. Karşımıza çıkan cmd penceresine aşağıdaki komutları yazıyoruz.

diskpart
list disk
select disk 1 (Anasistem olan Mac OS kurulu olmayan diski seçiyoruz.)
clean

Akabinde “Yenile” ye tıklayarak Windows kurulacak disk seçilir.

Windows kurulduktan sonra Boot Camp asistan yüklenir.

Windows ile Mac OS arasında geçiş yapmak için klavyeden Option ve R veya Alt ve R tuşuna basıyoruz.
