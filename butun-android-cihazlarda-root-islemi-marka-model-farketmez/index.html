Bütün android cihazlarda root işlemi [Marka model farketmez]

11 Şubat 2021 0 Yazar: Yakup CONTARLI

Merhaba Muhterem Ziyaretçilerimiz,

Bu yazımızda Marka Model farketmeksizin bütün android cihazlarınıza root işlemi nasıl yapılacağı hususnda olacak.

Bunun için Magisk Manager’i yüklemek gerekmektedir.

Magisk Manager

Magisk Manager ile yükleme aşamaları

Cihazınızın rom dosyasını bulup boot.img dosyasını telefonunuza atınız.

Magisk Manager’, açıp yükle\İnstall’e dokununuz.

ABV 2.0/dm-verity’i koru işaretlemeden “İleri” dokunuyoruz.

“Dosya seçin ve yamalayın” seçeneğini seçip karşımıza açılan pencereden cihazımıza aktardğımız boot.img dosyasını seçiyoruz.

“Devam Et” dokunarak patch işlemini başlayacak. “All done” ibraseni gördükten sonra (/scdard/Download) dizini veya Downloads klasöründeki bilgisayarınıza patched… ile başlayam img dosyasını aktarınız.

Bilgisayarınıza atardıktan sonra dosyanın ismini boot.img olarak değiştiriniz.

Sonra aşağıdaki linkteki işlemi yapınız.

https://bilisimkurdu.github.io/android-cihazlarda-oem-kilidi-acmak/

Sonra cd komutu ile boot.img dosyasının bulunduğu dizine gelip aşağıdaki komutları yazınız.

fastboot flash boot boot.img
fastboot reboot

İşlemimiz tamamlanmıştır.
