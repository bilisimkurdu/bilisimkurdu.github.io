Debian tabanlı işletim sistemlerinde depolarda bulunmayan paketleri yüklemek

Yayın tarihi 28 Kasım 2019Yayınlayan Yakup CONTARLI

Merhaba Muhterem Ziyaretçiler,

Bu yazımda debian tabanlı işletim sistemlerinin depolarında bulunmayan eksik paketlerin yüklenmesiyle ilişkin olacak.

Eksik paketler nasıl yüklenir?

/etc/apt/sources.list dosyasını root hakları ile açıp aşağıdaki satırı ekleyiniz.

deb http://debian.opennms.org/ stable main

Sonra aşağıdaki komutları giriniz.

sudo wget -O - http://debian.opennms.org/OPENNMS-GPG-KEY | sudo apt-key add -
sudo apt-get update
sudo apt-get install paketAdi

Eğer deb uzantılı dosyaları yüklerken almışsanız.

cd dosyaDizini
sudo apt-get install gdebi
sudo gdebi paketAdi.deb

komutların girilmesi yeterlidir.

Eğer bu yazıyla ilişkin anlatamadığım\anlaşılamayan konular varsa yazının altına yorum yapabilirsiniz.

İyi günler.

debian not package repository debian paket debian paket yüklenmiyor not package repository pardus depolarında paket yok pardus paket pardus paket yüklenmiyor ubuntu not package repository ubuntu paket ubuntu paket yüklenmiyor
