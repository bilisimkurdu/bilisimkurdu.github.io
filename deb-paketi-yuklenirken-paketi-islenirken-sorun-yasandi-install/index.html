deb paketi yüklenirken “paketi işlenirken sorun yaşandı (–install)”

7 Ekim 2019 0 Yazar: Yakup Contarlı

Merhaba Muhterem Ziyaretçiler,

.deb uzantılı paketleri yüklerken bazı problemler meydana gelmektedir. Bu yazımda da yaşanan sorunların birinden bahsedeceğim.

paketi işlenirken sorun yaşandı (–install): hatası bağımlılık gerektiren paketlerin bilgisayarda yüklü bulunmamasından kaynaklanmaktadır.

Sorunun çözümü için aşağıdaki komutlar sırasıyla girilir:

sudo apt-get update -y
sudo apt-get upgrade -y
sudo apt --fix-broken install

Akabinden .deb paketi tekrardan yüklenir.
